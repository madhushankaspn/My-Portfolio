## 🚀 Getting Started

To run this project locally, follow these steps:

1.  **Clone the Repo:**
    ```bash
    git clone [https://github.com/madhusankaspn/my-portfolio.git](https://github.com/madhusankaspn/my-portfolio.git)
    ```
2.  **Install Dependencies:**
    ```bash
    npm install
    ```
3.  **Run Development Server:**
    ```bash
    npm run dev
    ```
